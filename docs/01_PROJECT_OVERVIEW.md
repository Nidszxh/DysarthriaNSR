## 1. Project Summary

| Field | Value |
|-------|-------|
| Goal | Neuro-symbolic ASR for dysarthric speech; phoneme-level recognition with clinical explainability |
| Dataset | TORGO (15 speakers: F01, F03, F04, M01–M05 dysarthric; FC01–FC03, MC01–MC04 control) |
| Backbone | `facebook/hubert-base-ls960` (HuBERT-base, 94.7M params) |
| Additional modules | `SeverityAdapter`, `ArticulatoryFeatureEncoder`, `LearnableConstraintMatrix`, `SymbolicConstraintLayer` |
| Framework | PyTorch Lightning 2.x |
| Precision | `bf16-mixed` (RTX 4060 8GB target) |
| Tracking | MLflow (`mlruns/`) + checkpoints (`checkpoints/<run_name>/`) |
| Results | `results/<run_name>/evaluation_results.json` + companion plots |
| Target venue | SPCOM 2026 |

Phonemes use **stress-agnostic ARPABET**. `normalize_phoneme()` (in `src/utils/config.py`) strips stress digits. Special token IDs: `<BLANK>`=0, `<PAD>`=1, `<UNK>`=2.

## 3. Model Architecture

```
Input audio [B, T_audio]
  │
  ▼
HuBERT (facebook/hubert-base-ls960)
  94.68M params; CNN feature extractor (frozen) + 12 transformer layers
  Freeze schedule: permanently [0–3], progressive [4–11] via 3-stage unfreezing
  Output: hidden_states [B, T_h, 768]   where T_h ≈ T_audio / 320
  │
  ▼ (if use_severity_adapter=True)
SeverityAdapter
  severity [B] → Linear(1→64) → SiLU → Linear(64→768) → [B, 1, 768]
  cross-attention: Q=hidden_states, K=V=severity_ctx → residual + LayerNorm
  Note: For control speakers (severity=0), the adapter still contributes a learned
  constant offset (bias), not zeros — correct for both populations.
  │
  ▼ (if use_spec_augment=True)
SpecAugmentLayer
  Per-sample independent masks (per-sample loop, B13 fix applied)
  │
  ▼ (if use_temporal_downsample=True)
TemporalDownsampler
  stride-2 Conv1d(768, 768, kernel=3, stride=2, padding=1) → LayerNorm → GeLU
  T_h → T_h/2  (~25 Hz)
  │
  ▼
PhonemeClassifier
  768 → LayerNorm → GeLU → Dropout(0.1) → Linear(768→512) → GeLU → Linear(512→|V|=47)
  Output: logits_neural [B, T, 47]
  │
  ├── Articulatory heads (utterance-level via GAP then Linear):
  │     manner_head, place_head, voice_head
  │
  ▼
SymbolicConstraintLayer
  ├── LearnableConstraintMatrix:
  │     logit_C [47, 47] = log(C_static + ε) / 0.5  [temperature-sharpened; learnable]
  │     P_constrained = softmax(logit_C) @ P_neural
  │
  ├── Adaptive β:  β_adaptive = clamp(β_base + 0.2 * severity/5, 0, 0.8)
  │     Control speakers (severity=0): β ≈ 0.05
  │     Severe dysarthric (severity=5): β ≈ 0.25
  │
  └── P_final = β * P_constrained + (1-β) * P_neural  →  log(P_final + ε)
        Output: log_probs_constrained [B, T, 47]
```

### Ablation Modes

| Mode | CTC | CE | Art | Ordinal | BlankKL | SymKL | Constraint Layer |
|------|-----|----|-----|---------|---------|-------|-----------------|
| `full` | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ | symbolic blend |
| `neural_only` | ✅ | ✅ | ✅ | ✅ | ✅ | ❌ | bypassed |

| `symbolic_only` | ❌ | ❌ | ✅ | ❌ | ✅ | ✅ | symbolic blend only |

## 4. Multi-Task Loss

| Loss term | Formula | λ (current) | Notes |
|-----------|---------|-------------|-------|
| CTC | CTCLoss(log_probs_constrained, labels) | 0.80 | zero_infinity=True |
| Frame-CE | NLLLoss(log_probs_neural, padded_labels) | **0.10** (reduced from 0.35) | Labels pad/truncated — see §12 known issues |
| Articulatory CE | CE(manner)+CE(place)+CE(voice) / 3 | 0.08 | Utterance-level via GAP |
| Ordinal Contrastive | Pairwise cosine with ordinal margin | 0.05 | On last hidden states |
| Blank Prior KL | KL(Bernoulli(p_blank) ∥ Bernoulli(0.75)) | staged 0.10→0.20→0.35 | Target prob = 0.75 |
| Symbolic KL anchor | KL(C_learned ∥ C_static) | 0.50 | Prevents constraint matrix drift |

**Total**: `loss = 0.8·CTC + 0.10·CE + 0.08·art + 0.05·ordinal + λ_bkl·BlankKL + 0.50·SymKL`

## 5. Training Schedule

| Stage | Epoch trigger | HuBERT layers unfrozen |
|-------|--------------|------------------------|
| 0 — warmup | Epoch 0 | None (layers 0–11 all frozen) |
| 1 | Epoch 1 | 8, 9, 10, 11 |
| 2 | Epoch 6 | 6, 7, 8, 9, 10, 11 (direct call; no longer a no-op) |
| 3 | Epoch 12 | 4, 5, 6, 7, 8, 9, 10, 11 |

Optimizer: AdamW + OneCycleLR
LR groups: HuBERT ×0.1 | classifier+SeverityAdapter ×1.0 | SymbolicConstraintLayer ×0.5
Effective batch: 8 × 4 accumulation = 32

## 6. Evaluation Pipeline

- **Metrics**: Macro-speaker PER, WER, articulatory accuracy (manner/place/voice)
- **Statistics**: Bootstrap 95% CI (1000 samples), Welch t-test, Wilcoxon, Holm-Bonferroni
- **Decoding**: Greedy CTC collapse (primary), CTC prefix beam search (25-wide) + BigramLM fusion
- **Explainability**: PhonemeAttributor, ArticulatoryConfusionAnalyzer, ExplainableOutputFormatter → `explanations.json`
- **Uncertainty**: MC-Dropout (UncertaintyAwareDecoder, n\_samples=10); requires `--uncertainty` flag

## 8. LOSO-CV Guide

Leave-One-Speaker-Out (LOSO) cross-validation is the statistically valid evaluation strategy for TORGO. With 15 speakers, a single split gives unreliable estimates. LOSO rotates through all speakers, trains 15 independent models, and averages metrics.

**TORGO speakers:**
- Dysarthric: M01, M02, M03, M04, M05, F01, F03, F04
- Controls: MC01, MC02, MC03, MC04, MC05, FC01, FC03

### Running LOSO

```bash
# Full 15-fold sweep (recommended, ~30h on RTX 4060)
python run_pipeline.py --run-name loso_v1 --loso

# Manual single fold
python run_pipeline.py --run-name loso_M01 --loso-speaker M01 --beam-search --beam-width 25 --explain

# Loop all folds manually
for SPEAKER in M01 M02 M03 M04 M05 F01 F03 F04 MC01 MC02 MC03 MC04 MC05 FC01 FC03; do
    python run_pipeline.py --run-name "loso_${SPEAKER}" --loso-speaker "${SPEAKER}" \
        --beam-search --beam-width 25 --explain
done
```

### Expected Runtime

| Stage | Time (RTX 4060 8GB) |
|-------|----------------------|
| Per-fold training (30 epochs) | ~2h |
| Per-fold evaluation (beam w=25) | ~10min |
| Full 15-fold sweep | Completed (`loso_v1`, 15/15 folds) |

### Aggregating Results

```python
import json, pathlib, numpy as np

per_scores = []
for fold_dir in sorted(pathlib.Path("results").glob("loso_*")):
    results_file = fold_dir / "evaluation_results.json"
    if not results_file.exists():
        continue
    with open(results_file) as f:
        data = json.load(f)
    per_scores.append(data["per_beam"])

per_scores = np.array(per_scores)
macro_per = per_scores.mean()
ci_lo, ci_hi = np.percentile(
    [np.random.choice(per_scores, len(per_scores)).mean() for _ in range(5000)],
    [2.5, 97.5]
)
print(f"Macro PER = {macro_per:.4f}  95% CI [{ci_lo:.4f}, {ci_hi:.4f}]  n={len(per_scores)}")
```

### Statistical Tests

```python
from scipy import stats

dysarthric_speakers = ["M01","M02","M03","M04","M05","F01","F03","F04"]
control_speakers    = ["MC01","MC02","MC03","MC04","MC05","FC01","FC03"]

dys_per = [per_by_speaker[s] for s in dysarthric_speakers]
ctl_per = [per_by_speaker[s] for s in control_speakers]

t_stat, p_val = stats.ttest_ind(dys_per, ctl_per, equal_var=False)  # Welch t-test

severity = [get_speaker_severity(s) for s in all_speakers]
pearson_r, pearson_p = stats.pearsonr(severity, per_scores)
```

### Interpreting Results

| Metric | Interpretation |
|--------|----------------|
| Macro-PER | Average PER treating each speaker equally |
| Dysarthric PER | Mean of 8 dysarthric speakers' test PER |
| Control PER | Mean of 7 control speakers' test PER |
| Pearson r (severity–PER) | Expect positive correlation (higher severity → higher PER) |
| Wilcoxon p (dys vs ctrl) | Non-parametric; more robust given n=8 vs n=7 |

### Reproducibility

Each fold saves a `config.yaml` alongside its checkpoint. To reproduce fold M01:

```bash
python run_pipeline.py --run-name loso_M01_repro \
    --config results/loso_M01/config.yaml --loso-speaker M01
```

## 12. Known Limitations & Open Issues

### Critical (Publication-Blocking)

| Issue | Detail | Status |
|-------|--------|--------|
| Dysarthric fold variance remains high | Several dysarthric held-out folds remain high-PER with insertion-heavy errors despite LOSO completion (notably M01/M02/M04/M05/F01). | ⚠️ Active post-LOSO optimization target |
| Frame-CE label alignment (T-05) | `align_labels_to_logits` pads/truncates; supervises only first L frames without real phoneme-to-frame alignment. Mitigation: lambda\_ce reduced 0.35→0.10 (C-1). | ⚠️ Mitigated, not fixed |
| Small-split statistical fragility | Single-split statistics are fragile; publication reporting now uses completed LOSO (`loso_v1`, 15 folds). | ✅ Addressed via LOSO |

### Major

| Issue | Detail | Status |
|-------|--------|--------|
| `OrdinalContrastiveLoss` zero-margin for control pairs | Mitigated by continuous `TORGO_SEVERITY_MAP` usage in train/eval forward path. | ✅ Fixed in active path |
| `BigramLMScorer` zero-count transitions | Add-k smoothing prevents `log(0)` hypothesis collapse. | ✅ Fixed |
| Paired significance test (neural vs constrained) | Bootstrap paired Δ PER test and p-values are now reported. | ✅ Fixed |

### Minor

| Issue | Detail | Status |
|-------|--------|--------|
| `_PROBABLE_CAUSE_MAP` key format mismatch in `attribution.py` | Key formats were aligned; fallback no longer collapses to universal unknown path. | ✅ Fixed |
| `rule_precision()` never called | Implemented in `rule_tracker.py` but not wired into `evaluate_model` output. | ❌ Open |
| `PHONEME_DETAILS` / `PHONEME_FEATURES` duplication | Defined separately in `manifest.py` and `model.py`; must be kept in sync manually. Recommendation: move to `src/utils/constants.py`. | ❌ Open |
| `ModelConfig.num_phonemes=44` misleading | Actual vocab size is 47 (44 ARPABET + 3 special tokens). | ❌ Minor |
| `BlankPriorKLLoss` class default mismatch | Defaults/tests aligned to target blank probability 0.75. | ✅ Fixed |
| `conformal_phoneme_sets` τ tautology | `tau = 1 - (1 - coverage) = coverage`; not calibrated conformal prediction. Documented as APS-like heuristic (M-5). | ✅ Documented |
| `PhonemeAttributor.attention_attribution` not implemented | Requires CTC forced alignment (future work). | ⚠️ Known gap |
| Uncertainty always null unless `--uncertainty` flag | MC Dropout inference not default. | By design |
